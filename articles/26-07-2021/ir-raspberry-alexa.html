Introducción y tecnologías

El objetivo de este proyecto era construír un mando universal para controlar por voz una SmartTV de Samsung. La propia SmartTV consta de una Skill oficial para Alexa, pero dicha aplicación está bastante limitada permitiendo únicamente operaciones sencillas como subir o bajar el volumen y cambiar de canal.

Para superar estas limitaciones surge la idea de iniciar un proyecto con una Raspberry PI 4 que tenía abandonada. Este proyecto toma como punto de partida este tutorial, donde se explica todo el proceso desde el montaje del circuito hasta la integración con Alexa. Aunque, en este caso tanto la construcción del circuito como la propia implementación del Skill de Alexa difieren bastante. Por una parte, se utilizó ExpressJS y el adaptador del SDK de Skills de Alexa (ask-sdk-express-adapter). Por otra parte, el procedimiento para grabar las señales de infrarrojo y emitirlas es completamente diferente, puesto que el software propuesto para capturar y emitir señales de IR (lirc) no funcionaba apropiadamente en el sistema operativo actualizado de mi Raspberry.

Este proyecto no solo incluye software, sino que también se necesita montar el circuito de infrarrojos. En mi caso, que tengo 0 experiencia con estos temas, ha resultado bastante complejo inicialmente. He llegado a montar el circuito innumerables veces hasta darme cuenta de que el problema estaba en el propio software. Detallo a continuación todos los componentes hardware y el software necesario para ser lo más preciso posible y evitar posibles problemas.

Hardware
  • Raspberry PI 4 (en mi caso con la versión 5.10.17-v7l del kernel)
  • Breadboard (solo se necesita una, yo he pedido varias por si quiero hacer otros proyectos)
  • Resistencias (recomendable pedir varias por si acaso)
  • Cables (importante tener cables macho-macho y macho-hembra)
  • Diodos IR (solo he utilizado los emisores)
  • Receptor IR (solo se necesita uno)
  • Transistor (solo se necesita uno)
  • Echo Dot
Software
  • ExpressJS (para montar el servicio que expone las acciones o Intents de la Skill de Alexa)
  • ask-sdk-express-adapter (para construir los propios Intents de la Skill de Alexa)
  • Python (para construir un script muy sencillo que permita capturar y emitir infrarrojos mediante el circuito)
  • Cuenta de desarrollo en Amazon (para crear la Skill de Alexa)
  • ngrok (servicio que permite tunelizar un servicio en local a través de Internet, para un entorno más estable se puede pagar por este servicio)
